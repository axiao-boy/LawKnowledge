> 法律知识库 - 劳动法


## 📚 劳动法

《中华人民共和国劳动法》是调整劳动关系以及与劳动关系密切联系的社会关系的法律规范总称。

## 📖 劳动法概述

劳动法是保障劳动者合法权益、规范用人单位用工行为的基本法律。其核心内容包括：

- **就业促进**：国家促进就业的政策和措施
- **劳动合同**：劳动合同的订立、履行、变更、解除和终止
- **工作时间和休息休假**：法定工作时间、休息时间、节假日等
- **工资**：工资分配原则、最低工资标准等
- **劳动安全卫生**：劳动安全卫生设施和条件
- **社会保险和福利**：社会保险的种类和待遇
- **劳动争议处理**：劳动争议的解决方式和程序

## 📚 劳动合同法

《中华人民共和国劳动合同法》是为了完善劳动合同制度，明确劳动合同双方当事人的权利和义务，保护劳动者的合法权益，构建和发展和谐稳定的劳动关系而制定的法律。

## 📖 劳动合同法概述

劳动合同法是规范劳动合同的订立、履行、变更、解除和终止等行为的基本法律。其核心内容包括：

- **劳动合同的订立**：书面劳动合同的签订、必备条款与约定条款
- **试用期**：试用期期限、工资标准及解除条件
- **服务期与培训**：专项培训费用与服务期约定、违约金责任
- **保密与竞业限制**：商业秘密保护、竞业限制的范围与期限
- **劳动合同的履行与变更**：履行原则、变更的协商一致要求
- **劳动合同的解除和终止**：协商解除、法定解除、用人单位单方解除与劳动者单方解除
- **经济补偿与赔偿金**：经济补偿金的计算标准、违法解除的赔偿金
- **劳务派遣**：派遣单位与用工单位的义务、同工同酬原则
- **非全日制用工**：用工形式、计酬方式与终止规则
- **集体合同**：集体协商、集体合同的订立与效力
