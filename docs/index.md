## 📚 劳动法

《中华人民共和国劳动法》是调整劳动关系以及与劳动关系密切联系的社会关系的法律规范总称。

## 📖 劳动法概述

劳动法是保障劳动者合法权益、规范用人单位用工行为的基本法律。其核心内容包括：

- **就业促进**：国家促进就业的政策和措施
- **劳动合同**：劳动合同的订立、履行、变更、解除和终止
- **工作时间和休息休假**：法定工作时间、休息时间、节假日等
- **工资**：工资分配原则、最低工资标准等
- **劳动安全卫生**：劳动安全卫生设施和条件
- **社会保险和福利**：社会保险的种类和待遇
- **劳动争议处理**：劳动争议的解决方式和程序
